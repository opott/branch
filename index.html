The collaboration challenge from Hack Club

How does this work?

Branch is a collaboration challenge designed to teach you the essentials of Git and GitHub for collaborative coding.

Groups will start with an initial workshop where they reuse skills from Boba Drops to create a page for their group's website. Participants will learn how to contribute their changes back to the base repository for the updates to be reflected on the website.

Following completion of the initial challenge, groups will be invited to further collaborative software and hardware challenges.

Who is this for?

Branch primarily targets high school Hack Clubs, but we also encourage groups of friends to participate.
The only requirement is that all participants are aged 18 or under.

What's in it for me?

There will be a reward for all participants who have an approved submission for the initial workshop. This is currently undecided, but is likely to be Hack Club swag or an electronics kit.
